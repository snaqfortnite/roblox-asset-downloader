# Roblox Asset Downloader

A lightweight, browser-based tool for downloading raw Roblox assets and converting them into usable files like images or audio.

## Features

- Download raw Roblox assets using an Asset ID
- Convert raw files into usable formats
- Supports:
  - Decals (`.png`)
  - Audio (`.mp3`)
- Drag & drop support
- Manual file picker support
- Fully client-side (no servers, no installs)

---

## How to Use

1. Open the site  
   https://snaqfortnite.github.io/roblox-asset-downloader/

2. Enter a Roblox **Asset ID**

3. Click **Download Raw**  
   - This opens the raw asset from Roblox in a new tab

4. Download the file from the Roblox page

5. Either:
   - Drag & drop the downloaded file into the drop area  
   **OR**
   - Click **Open File** and select the file manually

6. Choose the correct asset type (`.png` or `.mp3`)

7. The file will automatically download with the correct extension
